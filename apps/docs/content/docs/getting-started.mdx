## Installation

qrdx requires Node.js 18+ and works with React 18+.

### Using npm

```bash
npm install qrdx@latest
```

### Using yarn

```bash
yarn add qrdx@latest
```

### Using pnpm

```bash
pnpm add qrdx@latest
```

## Basic Setup

After installation, you can start using qrdx in your React components:

```tsx
import { QRCode } from 'qrdx';

export default function App() {
  return (
    <QRCode
      value="https://example.com"
      size={256}
    />
  );
}
```

## Requirements

- **Node.js**: 18.0.0 or higher
- **React**: 18.0.0 or higher (with React DOM)
- **TypeScript**: Optional but recommended for full type safety
