# qrdx Documentation

A powerful, flexible React component library for generating customizable QR codes with extensive customization options, templates, and server-side rendering support.

## Quick Start

```package-install
qrdx@latest
```

```tsx
import { QRCode } from 'qrdx';

export default function App() {
  return (
    <QRCode
      value="https://example.com"
      size={512}
      fgColor="#000000"
      bgColor="#ffffff"
    />
  );
}
```
