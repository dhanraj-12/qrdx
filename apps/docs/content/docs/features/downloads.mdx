# QR Code Downloads

qrdx provides comprehensive download functionality for exporting QR codes in multiple formats. Download your QR codes as PNG, JPG, SVG, or PDF with custom sizes and filenames.

## Quick Download

### Basic Download Function

The simplest way to download a QR code:

```tsx
import { downloadQRCode, PRESET_SIZES } from 'qrdx';

async function downloadBasicQR() {
  try {
    await downloadQRCode(
      {
        value: "https://example.com",
        size: 1024,
        fgColor: "#000000",
        bgColor: "#ffffff",
      },
      {
        format: "png",
        size: PRESET_SIZES.large, // 800x800px
        filename: "my-qrcode.png"
      }
    );
    console.log("Download successful!");
  } catch (error) {
    console.error("Download failed:", error);
  }
}
```

## Supported Formats

### PNG (Portable Network Graphics)

Best for web use, supports transparency, high quality:

```tsx
import { downloadQRCode } from 'qrdx';

await downloadQRCode(
  { value: "https://example.com", size: 512 },
  {
    format: "png",
    size: { width: 512, height: 512 },
    filename: "qrcode.png"
  }
);
```

### JPG/JPEG (Joint Photographic Experts Group)

Smaller file sizes, solid backgrounds only:

```tsx
import { downloadQRCode } from 'qrdx';

await downloadQRCode(
  { value: "https://example.com", size: 512 },
  {
    format: "jpg",
    size: { width: 512, height: 512 },
    filename: "qrcode.jpg"
  }
);
```

### SVG (Scalable Vector Graphics)

Perfect for print, infinitely scalable, vector format:

```tsx
import { downloadQRCode } from 'qrdx';

await downloadQRCode(
  { value: "https://example.com", size: 512 },
  {
    format: "svg",
    size: { width: 512, height: 512 },
    filename: "qrcode.svg"
  }
);
```

### PDF (Portable Document Format)

Ideal for professional documents and high-quality printing:

```tsx
import { downloadQRCode } from 'qrdx';

await downloadQRCode(
  { value: "https://example.com", size: 1024 },
  {
    format: "pdf",
    size: { width: 1024, height: 1024 },
```

## Preset Sizes

qrdx includes convenient preset sizes for common use cases:

```tsx
import { PRESET_SIZES } from 'qrdx';

console.log(PRESET_SIZES.small);   // { width: 200, height: 200 }
console.log(PRESET_SIZES.medium);  // { width: 400, height: 400 }
console.log(PRESET_SIZES.large);   // { width: 800, height: 800 }
console.log(PRESET_SIZES.xlarge);  // { width: 1200, height: 1200 }
console.log(PRESET_SIZES["2xl"]);  // { width: 1600, height: 1600 }
console.log(PRESET_SIZES["3xl"]);  // { width: 2000, height: 2000 }
```

### Size Recommendations

- **Small (200px)**: Quick sharing, social media thumbnails
- **Medium (400px)**: Standard web use, documents
- **Large (800px)**: High-quality web images
- **X-Large (1200px)**: Print materials, posters
- **2X-Large (1600px)**: Large format printing
- **3X-Large (2000px)**: Maximum quality printing

### Size Validation

qrdx validates download sizes for quality:

```tsx

const validation = validateSize(512, 512);
if (validation.isValid) {

  console.error("Invalid size:", validation.error);
```

## Batch Downloads

### Multiple QR Codes

Download multiple QR codes simultaneously:

```tsx
import { downloadQRCode } from 'qrdx';

async function batchDownload() {
  const qrCodes = [
    {
      props: { value: "https://example.com", size: 512 },
      options: { format: "png", size: { width: 512, height: 512 }, filename: "site-qr.png" }
    },
    {
      props: { value: "https://github.com/user/repo", size: 512 },
      options: { format: "svg", size: { width: 512, height: 512 }, filename: "github-qr.svg" }
    },
    {
      props: { value: "tel:+1234567890", size: 512 },
      options: { format: "pdf", size: { width: 1024, height: 1024 }, filename: "phone-qr.pdf" }
    }
  ];

  for (const qr of qrCodes) {
    try {
      await downloadQRCode(qr.props, qr.options);
      console.log(`Downloaded ${qr.options.filename}`);
    } catch (error) {
      console.error(`Failed to download ${qr.options.filename}:`, error);
    }
  }
}
```

### Dynamic Filenames

Generate filenames based on content or date:

```tsx
import { downloadQRCode } from 'qrdx';

function generateFilename(value: string, format: string): string {
  // Create readable filename from URL or text
  const sanitized = value
    .replace(/^https?:\/\//, '') // Remove protocol
    .replace(/[^a-zA-Z0-9]/g, '-') // Replace special chars
    .substring(0, 30); // Limit length

  const timestamp = new Date().toISOString().split('T')[0];
  return `${sanitized}-${timestamp}.${format}`;
}

// Usage
const filename = generateFilename("https://example.com", "png");
// Output: "example-com-2024-01-15.png"
```

## Custom Download Implementation

### Custom Download Logic

Implement your own download handling:

```tsx
import { getQRCodeDataUrl } from 'qrdx';

function customDownload(dataUrl: string, filename: string, mimeType?: string) {
  const link = document.createElement('a');
  link.href = dataUrl;
  link.download = filename;
  link.style.display = 'none';

  // Optional: set MIME type
  if (mimeType) {
    link.type = mimeType;
  }

  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
}

async function customDownloadExample() {
  const dataUrl = await getQRCodeDataUrl(
    { value: "https://example.com", size: 512 },
    "png",
    { width: 512, height: 512 }
  );

  customDownload(dataUrl, "custom-qr-code.png", "image/png");
}
```

## Error Handling

### Comprehensive Error Handling

```tsx
import { downloadQRCode } from 'qrdx';

async function safeDownload() {
  try {
    await downloadQRCode(
      {
        value: "https://example.com",
        size: 512,
        // Ensure all required props are provided
      },
      {
        format: "png",
        size: { width: 512, height: 512 },
        filename: "safe-download.png"
      }
    );

    console.log("Download successful!");
  } catch (error) {
    // Handle different error types
    if (error instanceof Error) {
      switch (true) {
        case error.message.includes('size'):
          alert('Invalid size specified. Please check dimensions.');
          break;
        case error.message.includes('network'):
          alert('Network error. Please check your connection.');
          break;
        case error.message.includes('canvas'):
          alert('Canvas rendering error. Try a different format.');
          break;
        default:
          alert(`Download failed: ${error.message}`);
      }
    } else {
      alert('An unexpected error occurred during download.');
    }
  }
}
```

## Performance Considerations

### Format Selection

- **PNG**: Best quality, larger files, supports transparency
- **JPG**: Smaller files, no transparency, good for photos
- **SVG**: Scalable, small files, perfect for web/print
- **PDF**: Professional, good for documents, larger files

### Size Optimization

- **Web use**: 256-512px, PNG format
- **Print**: 1024px+, PDF or high-res PNG
- **Mobile**: 256px, optimized for fast loading
- **Email**: 200-300px, balance size/quality

## Browser Compatibility

Downloads work in modern browsers with these requirements:

- **File API**: For file creation and download
- **Canvas API**: For PNG/JPG generation
- **Blob API**: For clipboard operations
- **ES6+**: Modern JavaScript features
